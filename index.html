Cara Memulai ExRestoPro

  1. 1
    Klik link & beli aplikasi melalui App Store
    Cari ExRestoPro di App Store, atau klik tombol download di halaman ini.
  2. 2
    Install aplikasi
    Setelah pembelian, install aplikasi. Otomatis terisi demo produk & data contoh.
  3. 3
    Login dengan PIN 1234
    PIN default: 1234. Anda bisa ganti PIN di menu pengaturan.
  4. 4
    Edit data usaha
    Masuk ke Settings → Store, isi keterangan usaha Anda (nama, alamat, dsb).
  5. 5
    Mulai shift & transaksi
    Buka menu Shift lalu Add Shift untuk mulai. Untuk transaksi, pilih Order → Add Order.
